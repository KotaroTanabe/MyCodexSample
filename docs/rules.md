# Rules Supported

現在実装されているルールは以下の通りです。

- リーチ (Reach)
- ダブルリーチ (Double Riichi)
- ドラ (Dora)
- 本場 (Honba)
- 裏ドラ (Ura Dora)
- 一発 (Ippatsu)
